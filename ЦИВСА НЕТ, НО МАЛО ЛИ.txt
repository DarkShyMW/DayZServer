Команды для настройки подключения к общей папки на SRV

1. На WEB-L и WEB-R

	apt install cifs-utils - установка инструментов для работы с SMB

	mount.cifs //SRV/Share /opt/share -o user=nologin,password= - команда для монтирования каталога. 
	#	//SRV - сервер с общей папкой, 
	#	/Share - название общей папки, 
	#	/opt/share - куда монтировать, 
	#	user=nologin,password = используем для подключения анонимного пользователя без пароля 

	ls /opt/share - смотрим, что каталог примонтировался. Для этого предварительно создайте на SRV в общей папке какой-нибудь файл или папку, вы должны их увидеть при монтировании

2. Проверяем работоспособность общей папки. Все созданные объекты мы должны видеть на SRV.

	В /opt/share:
	echo "Что-то" > какой-то_файл.txt - команда для создания файла с неким содержимым
	mkdir название_каталога - создаем каталог

3. Проверив, что каталог монтируется и есть все необходимые права, настраиваем автомонтирование

	vim /etc/fstab - открываем файл и добавляем строчку:

//SRV/Share	/opt/share	cifs	password=,noperm,_netdev	0	0

	#	umount /opt/share - размонтировать каталог
	#	mount -a - считываем данные из /etc/fstab, каталог должен примонтироваться автоматически без какого-либо вывода в консоль
	#	ls /opt/share - снова смотрим, что каталог примонтировался
